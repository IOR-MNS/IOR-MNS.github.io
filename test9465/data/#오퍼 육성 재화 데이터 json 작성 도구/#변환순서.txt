[input.txt 만들기]

rawdata.txt에 대해 선처리.txt에 명시된 대로 선처리 수행.
선처리된 파일에 대해 dic.csv를 적용. 이때, maptext.py를 활용
마지막으로, 적절하게 후처리를 수행하면 성공. (보유 스킬이 123으로 표시되어 있는걸 3으로, 12로 표기되어있는걸 2로 바꾸는 작업 등, json 변환 프로그램의 양식 준수를 위한 후처리 수행) (양식 준수에 관해서는, 원시데이터 변환 알고리즘.txt를 참고)


// 이미 input.txt가 완성되어 있고, 추가할 오퍼레이터가 적다면, 손수 input.txt에 양식을 지켜서 정보를 추가작성해도 된다.

input.txt를 ANSI 인코딩으로 변환한 후, 모든 개행문자('\r', '\n' 등)를 '\n'으로 통일한 후, 명일방주 오퍼 육성재료 json 변환기를 통해 json파일로 변환.
출력된 json파일을 다시 UTF-8 (BOM 없음) 인코딩으로 변환.

dic_itemData.csv, dic_opName.csv를 차례대로 적용.

일반 치환으로 '만'을 '0000'으로 치환하고,
정규식 치환으로 '([0-9])\.([0-9]000)0' 을 '$1$2'으로 치환.
