- 자신이 파밍하려는 등급에 해당하는 재료의 부족량만 확인하면 됨

- 화면 우측 하단의 메뉴 버튼을 눌러 표시/필터 옵션을 설정하거나, 편의 기능을 사용할 수 있음
- [기능]
- 부족한 아이템만 표시하는 기능
- 보유량-필요량 정리 기능
- 예) [보유량 7 / 필요량 4]에서 [보유량 3 / 필요량 0]으로 정리
- [옵션]
- 이름, 입력란 표시 여부를 설정 가능
- 재료의 등급, 카테고리별 필터 설정 가능
